## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm v6.14.8

- node v14.5.0

### Installation

1. Open cmd

2. Clone the repo

```sh
git clone https://github.com/de4th-zone/d4z-react-redux.git
```

3. Open folder

```sh
cd d4z-react-redux-master
```

4. Install NPM packages

```sh
npm install
```

4. Run project

```sh
npm run dev
```
